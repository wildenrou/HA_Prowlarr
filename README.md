# HA Prowlarr

Custom Home Assistant integration for Prowlarr.

## Features

- Indexer count sensors
- Healthy/unhealthy indexer sensors
- Torrent and usenet indexer counts
- Application and download client counts
- Health issue sensors
- Online status binary sensor

## Installation with HACS

1. HACS -> Integrations -> 3 dots -> Custom repositories
2. Add this repository URL
3. Category: Integration
4. Install "Prowlarr"
5. Restart Home Assistant
6. Add integration from Settings -> Devices & Services

## Configuration

You need:
- Host
- Port (default 9696)
- API key
- SSL on/off
